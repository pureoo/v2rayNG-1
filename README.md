# 功能  
1.服务器配置可以新增并编辑  
2.支持扫描二维码和导入剪贴板  
3.分享功能,可以分享二维码或者导出剪贴板  
4.去掉流量统计等部分功能  

不知道手机cpu架构则下载app-universal-release.apk
